# 阶段 1 · Score、能量模型与 Langevin 采样（约 3 天）

> 目标：理解 score 为什么能绕开"难算的归一化常数"，并掌握用 score 采样的核心引擎——Langevin dynamics。学完你能在 toy 分布上用真实 score 把粒子"走"到数据分布上。
> （梯度提示：这一阶段先用"已知的真实 score"做采样，把采样和"如何学 score"解耦，下一阶段才学怎么估计 score。）

主线资料：[Yang Song 博客 score 部分](https://yang-song.net/blog/2021/score/) + 任意一份 EBM/Langevin 入门笔记。

## 任务清单

### ☐ 1-1 能量模型 EBM 与配分函数的困境
- 写出能量模型形式：`p(x) = exp(-E(x)) / Z`，其中 `Z = ∫ exp(-E(x)) dx` 是配分函数。
- 关键观察：`Z` 是对全空间的积分，高维下根本算不出来，这正是最大似然训练 EBM 的核心障碍。
- **完成标志**：能解释为什么"想直接最大化 log p(x)"会卡在 `Z` 上。

### ☐ 1-2 ⭐ 看 score 如何让 Z 消失
- 本阶段核心洞察：对 EBM 取 score，`∇x log p(x) = ∇x (-E(x) - log Z) = -∇x E(x)`，因为 `log Z` 不含 x，求导后归零。
- 也就是说：**只要建模 score，就完全不用碰配分函数**。这是 score matching 整套理论的出发点。
- 思考题：如果两个分布只差一个常数倍（未归一化），它们的 score 一样吗？为什么这对采样无所谓？

### ☐ 1-3 推导 Langevin dynamics 的更新公式
- 写出更新式（朗之万采样）：`x ← x + (ε/2)·∇x log p(x) + √ε · z`，其中 `z ~ N(0, I)`，`ε` 是步长。
- 逐项理解：梯度项把粒子推向高密度区，噪声项 `√ε·z` 提供随机扰动让它探索、不塌缩到一个点。
- **完成标志**：能默写更新公式，并说清两项各自的作用。

### ☐ 1-4 代码：用"真实 score"在 toy 分布上跑 Langevin
- 取一个能写出解析 score 的分布（如混合高斯），实现 Langevin 采样并可视化粒子随迭代收敛：
  ```python
  def langevin(score_fn, x, steps=1000, eps=1e-2):
      for _ in range(steps):
          x = x + 0.5*eps*score_fn(x) + (eps**0.5)*torch.randn_like(x)
      return x
  ```
- 从随机初始点出发，跑若干步，画出最终粒子分布与目标分布对比。
- **完成标志**：粒子云最终覆盖目标分布的各个模式。

### ☐ 1-5 思考题 + 实验：噪声项到底有多重要
- 把上一步代码里的噪声项 `√ε·z` 去掉，只保留梯度上升，重新跑一遍。
- 观察现象：粒子会发生什么？（提示：所有粒子都奔向最近的众数/峰值，失去分布的多样性，变成"找极大值"而非"采样"）。
- 思考题：Langevin 采样和"梯度上升找 MAP"在数学上差在哪一项？
